# About V-Phone Project
V-Phone is a front-end website project for a smartphone selling company.  
This project was created for a Human Computer Interaction course, focusing on user interface design, usability, navigation flow, and responsive layout.

## Features
- Homepage
- About page
- Product page
- Promotion page
- Register page
- Responsive layout
- Interactive elements using JavaScript

## Tech Stack
- HTML
- CSS
- JavaScript

## HCI Aspects
This project applies several Human Computer Interaction principles, such as:
- Clear and simple navigation
- Consistent visual design
- Readable typography
- Organized product information
- User-friendly page layout
- Easy access to important pages
- Responsive interface for different screen sizes
